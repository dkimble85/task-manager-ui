# task-manager-ui

Client-side of task management application where users can create, edit, delete, and categorize tasks. The app features a dashboard with task statistics and supports client-side routing for different views (e.g., all tasks, completed tasks).

## Project Tech

This project includes the following libraries/tools:

* [x] React v19
* [x] TypeScript v5
* [ ] TanStack Query - data retrieval
* [ ] React Router - client-side router
* [ ] Zustand - state management
* [ ] Shad/cn - component library
* [ ] TailwindCSS - CSS styling library
* [x] Vite - bundling/build
* [ ] Playwright - end to end testing
